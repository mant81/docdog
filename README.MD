3️⃣ 구글 플레이 상세 설명 (복붙용)
DocDog(문서도그)는 PPT, 엑셀, TXT, PDF 등
다양한 문서를 빠르고 안전하게 열어볼 수 있는 문서 뷰어입니다.

• PPT / PPTX / XLS / XLSX / TXT / PDF 지원
• 별도 변환 없이 바로 열기
• Google Docs Viewer 기반 안정적인 렌더링
• 임시 업로드 + 자동 만료로 안전한 문서 관리
• 불필요한 파일은 자동 정리

DocDog는 문서를 저장하지 않고,
필요할 때 빠르게 보고 안전하게 관리합니다.
